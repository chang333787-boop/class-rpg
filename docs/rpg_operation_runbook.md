# 우리반 RPG — 교사용 운영/장애 대응 런북

> "앱이 안 돼요 / 데이터가 이상해요"가 생겼을 때 **교사가 바로 따라 할 수 있는 순서표**입니다.
> 기술 지식 없이도 되는 것 위주로, 위험한 조작은 ⚠️로 표시했습니다.

---

## 0. 가장 먼저 (거의 다 여기서 해결)

1. **새로고침** — 화면을 아래로 당기거나(태블릿) 브라우저 새로고침. 대부분의 "멈춤/안 보임"은 이걸로 해결.
2. **인터넷 확인** — 다른 사이트(포털 등)가 열리는지 확인. 학교 와이파이가 느리면 로딩이 오래 걸릴 수 있음.
3. **잠깐 기다리기** — 아침에 여러 명이 동시에 접속하면 30초~1분 느려질 수 있음.

> 키오스크(공용 태블릿)가 로딩에서 멈추면 이제 **"🔄 다시 시도" 버튼**이 뜹니다. 그걸 누르세요.

---

## 1. 증상별 대응

### "학생 화면이 안 열려요 / 로딩만 계속 돌아요"
- ① 새로고침 → ② 인터넷 확인 → ③ 다른 기기에선 되는지 확인.
- 특정 학생만 안 되면: 그 기기 문제(와이파이·시계). 시계가 틀리면 접속 시간(오전 8:30~오후 4:00) 밖으로 판정될 수 있음 → **기기 날짜·시간을 자동으로 설정**.
- 전체가 안 되면: 인터넷 또는 서버 문제. 아래 4번(서버 상태) 참고.

### "학생이 한 게 사라졌어요 / 보상이 안 들어왔어요"
- 대부분 **네트워크 순단으로 저장이 안 된 것**. 학생에게 다시 해보라고 안내.
- 교사 승인 보상이 안 들어갔으면: 활동 승인 탭에서 다시 승인. (같은 걸 두 번 승인하지 않도록 확인)
- 반복되면 아래 3번(백업)으로 최근 상태 확인.

### "키오스크가 옛날 걸 보여줘요 (최신 퀘스트가 안 떠요)"
- 공용 태블릿을 오래 켜두면 연결이 낡을 수 있음 → **키오스크 새로고침**.
- 하루 시작할 때 한 번 새로고침하는 습관 권장.

### "화면은 됐다는데 실제로 저장이 안 된 것 같아요"
- 네트워크가 불안정했을 가능성. 중요한 승인·설정은 **한 후에 새로고침해서 반영됐는지 눈으로 확인**.

---

## 2. 데이터 백업 (평소에 해두기 — 가장 중요)

> ⚠️ 현재 자동 백업은 **교사가 관리자에 로그인할 때** 생기고 **최근 7일치만** 보관됩니다.
> 방학이나 장기 미접속 때는 자동 백업이 안 생기니, **아래 수동 내보내기를 습관화**하세요.

1. 관리자 → **게임 설정 → 데이터 관리 → "데이터 내보내기"** → JSON 파일이 다운로드됨.
2. 그 파일을 **학교 계정 클라우드나 USB 등 앱 밖에** 보관. (앱/서버가 통째로 망가져도 이 파일이 있으면 복구 가능)
3. **권장 주기: 주 1회 + 방학 전 반드시 1회.**

---

## 3. 데이터 복구 (문제가 생겼을 때)

> ⚠️ 아래는 **되돌릴 수 없는 조작**입니다. 하기 전에 반드시 2번(내보내기)으로 현재 상태를 먼저 저장하세요.

- **날짜별 롤백**: 관리자 → 데이터 관리 → "날짜별 롤백" → 되돌릴 날짜 선택.
  - ⚠️ 선택 날짜 **이후의 모든 변경이 사라집니다.**
  - ⚠️ 감정기록·독서·주간다짐 등 **일부 데이터는 자동 백업에 포함되지 않으니**, 롤백으로도 복구가 안 될 수 있습니다. 그래서 2번 수동 내보내기가 중요합니다.
- **데이터 가져오기**: 내보내둔 JSON 파일로 복원. ⚠️ 현재 데이터를 덮어씁니다.

---

## 4. 서버(Firebase) 상태

- 전체 학급이 동시에 안 되고 인터넷은 정상이면 → 서버(Firebase) 문제일 수 있음.
- **무료 요금제라 한 달 전송량 한도**가 있습니다. 학기 중반 이후 갑자기 전 학급이 안 되면 한도 초과 가능성 → Firebase 콘솔에서 사용량 확인(관리자에게 문의).
- ⚠️ **보안 점검(교사 직접)**: Firebase 콘솔에서 데이터베이스 규칙(rules)이 "테스트 모드"(만료형·전체 공개)인지 한 번 확인하세요. 현재 앱은 이 규칙이 유일한 보호막입니다.

---

## 5. 배포/업데이트 후

- 코드 업데이트 후 학생 화면이 이상하면: **강제 새로고침**(캐시 비우기). 태블릿은 브라우저 캐시 삭제 후 재접속.
- 업데이트 직후 10분 정도는 옛 버전과 새 버전이 섞일 수 있음(정상). 잠시 후 새로고침하면 해결.

---

## 6. 하지 말아야 할 것

- ⚠️ "전체 데이터 초기화" / "전체 학생 수치 초기화"는 **되돌릴 수 없음**. 반드시 2번 내보내기 후에만.
- ⚠️ 학기 중 실수로 눌렀다면 즉시 3번(롤백)으로 최근 상태 복구 시도.
